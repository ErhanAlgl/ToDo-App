## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ErhanAlgl/ToDo-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Todo-App
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
